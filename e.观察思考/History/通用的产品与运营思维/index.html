通用的产品与运营思维

Posted on 2021-05-01 | In 观察思考 | | 次阅读

现代互联网企业尤其是国内互联网企业除了开发外还有 产品 和 运营 两个所谓的“文职岗位”，即开放给不想写代码的计算机专业学生和想转互联网行业的其他专业学生的岗位。
不过我认为产品和运营两个领域的能力对于像我这样技术一般般的同学来说还是很重要的，用产品思维去设计软件或服务，哪怕是一个对外的 API 接口；用运营能力去在合适的位置推广软件或服务，获得持续的反馈。
